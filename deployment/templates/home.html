Business Problem


The problem that we are dealing with is known as Grammer Error Correction Problem. As the name suggests the given an input sentence in false English (here we are dealing with English Language) we have to predict the correct sentence which interprets the same meaning as the input sentence.


Applications

1. The problem has a huge application in our day to day practices because writing text is quite a prominent way of sharing information. Considering the fact that time is valuable to everybody, the model could assist the writer to write efficiently in a reduces time and effort.
2. Moreover, there are not many individuals who a fluent with english language. These kinds of models make sure that language does not become a barrier in sharing ideas. This will make sure that the reader and the writer are both on the same page.

Performance Metric

The task would be handled as a sequenc to sequence problem therefore that the metric that we would be using here is BELU score.

BLUE Score

This is a metric which is used to compare two sentences and gives a score based on the matching words. The perfect match give tha score of 1 and perfect unmatct gives a value of 0.It is widely used due to is simplicity and coorelation with human evaluation.

Dataset

The problem of this kind requires a high quality datasets which is very rarely available. After the literature survey we have narrowed down the datasets to two sources:
1. Lang-8
2. NUS Social Media Text Normalization and Translation Corpus
